# Portfolio Website

This is the very website you are looking at right now! It serves as my digital garden and portfolio.

## Tech Stack

- **Framework**: Next.js (React)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **State Management**: React Hooks
- **Animations**: Framer Motion & CSS Animations
- **Hosting**: GitHub Pages

## Features

- **Dark Mode**: A system-aware dark mode.
- **Responsive Design**: Looks great on all devices.
- **MDX Blog**: A blog system powered by MDX.
- **Project Showcase**: A filtered list of my projects.
